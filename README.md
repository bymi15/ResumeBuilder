# ResumeBuilder

## Tech Stack

- React
- NextJS
- Typescript
- Supabase
- TailwindCSS
- Shadcn
- Tanstack Query
- Zod
- Zustand

## Commands

### Run Development Server

```bash
npm run dev
```

### Generate Supabase DB Type Definitions

```bash
npx supabase gen types typescript --project-id "${SUPABASE_PROJECT_ID}" --schema public > src/types/database.types.ts
```
